File Information

How Batch Analysis Works

TXT Files

Format Requirements:

  • Plain text format (.txt)
  • One text per line or paragraph
  • Each text should be at least 10 characters
  • Maximum 50 texts will be processed

Example content:

I love this product!
The service was terrible.
This is an okay experience.
CSV Files

Format Requirements:

  • Comma-separated values (.csv)
  • Text column will be auto-detected
  • Columns with avg length > 20 chars are considered text
  • Maximum 100 rows will be processed

Example structure:

review,rating
"Great product, love it!",5
"Not what I expected.",2
